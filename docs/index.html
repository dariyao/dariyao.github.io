Northwind Traders

End-to-End Dimensional Data Warehouse for Analytics

SQL BigQuery dbt Tableau Dimensional Modelling Kimball Methodology

Dashboard Preview

The Problem

Northwind Traders relied on a legacy on-prem MySQL database for both transactions and reporting, causing performance bottlenecks, slow reports, and limited scalability.

What I Built

As part of an Analytics Engineering Bootcamp, I rebuilt the Northwind OLTP into an analytics-ready warehouse in BigQuery and took things further by building self-initiated executive dashboards in Tableau.

  • Kimball bus matrix with 4 conformed dimensions (customer, employee, product, date) and 3 fact tables.
  • 4-layer architecture: Data Lake → Staging → DWH → OBT in BigQuery.
  • dbt transformations with Jinja macros and incremental models.
  • Data quality tests: not_null, unique, relationship.
  • Performance optimization with partitioning and clustering.

Dashboards

Sales Overview

KPIs, trends, agent performance, top products

Product Inventory

Stock health, coverage, reorder queue

Customer Overview

Order frequency, geo spread, employee breakdown

Result

A production-ready warehouse with clean dimensional models, automated dbt transformations, and executive-friendly dashboards, allowing stakeholders to analyze sales performance, monitor inventory health, and understand customer behaviour.

Coffee Sales Dashboard

Interactive Excel Dashboard with Advanced Formulas

Excel XLOOKUP Index-Match Pivot Tables Data Cleaning

Dashboard Preview

The Problem

Transform raw coffee sales data into an interactive dashboard that reveals trends and performance metrics at a glance.

What I Built

A comprehensive Excel dashboard combining advanced formula engineering with interactive controls.

  • Dynamic lookups using XLOOKUP and Index-Match formulas for flexible data retrieval.
  • Sales calculations with multiplication formulas and nested IF functions for multi-criteria logic.
  • Pivot tables for dimensional analysis with timeline and slicer controls for real-time exploration.
  • Careful formatting with date and number standards, conditional formatting, and visual hierarchies.

Result

An analytics-ready dashboard that demonstrates proficiency with Excel's features with special attention paid to visualization.
